AmbulEEG Firmware Flasher

Flash the latest AmbulEEG firmware to your board straight from this page — no software to install.

How to flash

  1. Use Google Chrome or Microsoft Edge on a desktop/laptop.
  2. Plug the AmbulEEG board into a USB port with a data cable.
  3. Put the board into flashing mode — see the required step below. This is mandatory.
  4. Click Install, then select the board's serial port.
  5. Wait for it to finish — do not unplug until it says complete.
REQUIRED — DO THIS FIRST

You MUST put the board into flashing (download) mode

The board will not flash unless it is in download mode. If you skip this, the installer just spins on “Connecting…” forever. Before clicking Install, do this button sequence on the board:

  1. Press and HOLD the BOOT button (don't let go yet).
  2. While still holding BOOT, press and release the RESET button.
  3. Now release the BOOT button.
  4. The board is now in flashing mode — click Install below right away.

Flashing log

When you click Install above, the live progress shows here — the same output a developer sees in the browser console. The flash is done when you see Wrote … at 0x10000 followed by Leaving… and Hard resetting…. Any problems appear as red lines.

Selecting the serial port

After you click Install, your browser asks which device to connect to. In that pop-up:

  1. Click the port — it appears as USB JTAG/serial debug unit (COM…) or Silicon Labs CP210x USB to UART Bridge (COM…) (the COM number will vary).
  2. Click Connect.

No port showing up? Install the USB driver

If the pop-up is empty or the board never appears as a COM port, you likely need the Silicon Labs CP210x VCP driver. This board uses a CP2102N USB-to-UART bridge — it's the only driver you should ever need.

  1. Download the driver: Silicon Labs CP210x VCP Drivers.
  2. Install it, then unplug and replug the board.
  3. Reload this page and click Install again.

Windows 10/11 often install this automatically. macOS usually needs no driver.

Still stuck on “Connecting…”? Repeat the BOOT + RESET sequence above, then click Install again. Make sure no other program (e.g. a serial monitor or IDE) has the port open.
