AmbulEEG Firmware Flasher

Flash the latest AmbulEEG firmware to your board straight from this page — no software to install.

How to flash
  1. Use Google Chrome or Microsoft Edge on a desktop/laptop.
  2. Plug the AmbulEEG board into a USB port with a data cable.
  3. Click Install below and select the board's serial port.
  4. Wait for it to finish — do not unplug until it says complete.

If nothing happens when you plug in, the board may need to be put in download mode: hold the BOOT button, tap RESET, then release BOOT, and click Install again.
